# Como gerar o PDF exigido pelo trabalho

O enunciado pede um PDF com o mesmo conteúdo principal do `README.md`, incluindo gráficos.

## Passo a passo simples

1. Execute o projeto:

```bash
python main.py
```

2. Abra o arquivo `README.md`.
3. Copie o conteúdo para o Google Docs ou Microsoft Word.
4. Insira manualmente as imagens de `outputs/figures/`.
5. Exporte como `docs/relatorio.pdf`.

## Dica para a apresentação

Durante a apresentação, rode ao vivo:

```bash
python main.py
```

Assim você mostra o treinamento acontecendo, como pede o enunciado.
